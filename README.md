# Repository-Scale-Automated-Bug-Detection-and-Repair-via-Layered-Context-Aware-LLMs
LLM-powered system for bug detection, repair, and cross-language translation with self-validating, context-aware debugging.

Developed a compiler-like, interactive system that leverages Large Language Models (LLMs) for automated bug detection, repair, and cross-language code translation (Python, Java, C++). The system uses layered context injection (bug-level, repository-level, and project-level knowledge), multi-agent collaborative debugging, and self-validation loops to generate reliable patches.

Integrated dynamic analysis, symbolic execution, automated test generation, and CI/CD workflows to verify fixes and reduce hallucinated code. The platform also supports security vulnerability repair, adaptive context selection, and repository-scale reasoning, enabling smarter, faster, and safer software maintenance.
